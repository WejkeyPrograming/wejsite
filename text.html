Texty. Aj toto je text kde je použitý style.css môžeme tak upravovať napríklad farbu alebo font


Teraz si urobíme napríklad font, farbu a velkosť




Takto bude vyzerať HTML

<html>
<head>
<title>Text</title>
</head>

<body>
<h1 class="text"> Text Na Test </h1>
</body>

</html>

Takto bude vyzerať CSS

⠀
⠀
.text {
font-family: 'PT Serif', serif;
font-size: 20px;
color: #ffffff;
} ⠀
⠀
⠀















Čo sú fonty?




Font textu máš na každom webe. Je to niečo ako "peknejší text". V krátkosti, je to lepší dizajn textu



Font size. Size je velkosť textu. Ćím menší size tým menší text.



A color je farba textu.
